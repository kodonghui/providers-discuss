# Public Package README Discussion Input

This directory is a repo-relative input bundle for reviewing and drafting the
public `providers-discuss` README. It is safe to use from a cloned repository
where host-local KDH run-state paths do not exist.

## Objective

Write the public `providers-discuss` README. Cover installation, input folder
preparation, provider/seat/round/agent configuration, manual/import execution,
and current live-dispatch maturity boundaries clearly.

## Paths From Repo Root

- Config: `examples/discussions/public-package-readme/public-package-readme.config.json`
- Input pack: `examples/discussions/public-package-readme/input-pack/input-pack.md`
- Raw sources: `examples/discussions/public-package-readme/sources/`

If a provider workspace cannot see host-local KDH paths, use these repo-relative
paths instead.
